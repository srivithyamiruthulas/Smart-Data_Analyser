# Smart Data Analyser

Smart Data Analyser is a lightweight AI-powered application that helps beginners quickly understand the contents of a dataset. By uploading a CSV file, users receive an instant preview, statistical summary, feature insights, and visual analysis.

---

## Overview

This project simplifies the initial stage of data analysis by automatically generating meaningful insights from raw datasets. It is designed for students, researchers, and beginners who want fast dataset understanding without writing analysis code.

---

## Features

- Upload and analyze CSV datasets
- Automatic statistical summary
- Missing value analysis
- Automatic prediction target detection
- Feature importance using machine learning
- Visualization of key features
- AI-generated dataset interpretation

---

## Technology Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Gradio

---

## Installation and Usage

### Clone the repository
```bash
git clone https://github.com/srivithyamiruthulas/Smart-Data_Analyser.git
```

### Navigate to project folder
```bash
cd Smart-Data_Analyser
```

### Install dependencies
```bash
pip install -r requirements.txt
```

### Run the application
```bash
python app.py
```

Open the generated Gradio link in your browser.
